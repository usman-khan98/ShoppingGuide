# ShoppingGuide
Ecommerce Web Scraping

Requirements
1. pip install selenium
2. compatiable selenium python chrome driver https://chromedriver.chromium.org/downloads

Instructions
1. open terminal
2. cd backend
3. cd Authentication-Site
4. node app.js

Frontend
1. cd Frontend
2. npm start

Admin
1. cd admin
2. npm start
